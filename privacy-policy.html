Privacy

Last updated: 2026-05-09

Breathe Deep is a static page. There are no accounts, no servers, and no analytics.

What is collected

Nothing is sent anywhere. The site does not run third-party scripts, ads, or trackers.

What is stored on your device

Your settings (chosen technique, duration, sound and haptics preferences, theme) and a small streak counter live in your browser's localStorage. They never leave your device. Clearing site data removes them.

Notifications

If you enable the daily reminder, the browser asks for permission and shows local notifications at the chosen hour while the page or installed app is reachable. No push server is involved.

Service worker / offline

A service worker caches the app shell so the site loads without a network. It only fetches files from this origin.

Hosting logs

Like any website, the host (e.g. GitHub Pages) sees standard HTTP request metadata. That is outside the control of this app.
